# Swag_Moor 專案說明

## 專案用途
此資料夾主要用於 **Demo 展示**，包含 Web 與 APP 畫面的呈現原型。
所有畫面一律以 **Web 方式呈現**（包含 APP 畫面模擬）。

## 開發規範

### Git 操作
- **禁止**在未經確認前自動 push 到 GitHub
- 每次完成任務後，**必須詢問使用者**是否要推到 git
- 若有任何改動推到 GitHub，需記錄到 MEMORY.md

### 新功能開發流程
1. 先進行**架構分析**，列出技術選型、資料夾結構、元件規劃
2. 與使用者**確認架構**後，再開始實作
3. 完成後詢問是否推送至 git

## 技術方向
- 所有畫面以 Web 技術呈現
- APP 畫面以 Web 模擬（如手機框架包裝）
